# Estoque Cars Front

Aplicacao front-end do sistema Estoque Cars, desenvolvida com React e Vite.

## Como rodar

```bash
npm install
npm run dev
```

Aplicacao local:

```text
http://localhost:5173
```

## Build

```bash
npm run build
npm run preview
```

## Scripts uteis

```bash
npm run lint
npm run lint:fix
npm run format
npm run format:write
npm run test
```
